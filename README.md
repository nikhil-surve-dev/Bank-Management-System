## Bank Management System

A console-based application developed in C++ using Object-Oriented Programming and File Handling.

### Features
- Create Account
- Deposit Money
- Withdraw Money
- View Account Details
- Data stored using files (.DAT)

### Technologies Used
- C++
- OOP Concepts
- File Handling

### How to Run
1. Compile using Turbo C++ / any C++ compiler
2. Run the program
3. Follow on-screen instructions
